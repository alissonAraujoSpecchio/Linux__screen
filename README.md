# Linux_screen
Examples of screen commands

## Oppening a session
```
screen -R name
```

## Accessing a session
```
screen -x name
```

## Some CTRL-a commands

Exit a session:
```
CTRL-a d
```

Destroy current session:
```
CTRL-a k
```

Split screen horizontally
```
CTRL-a S
```

## List screens
```
screen -list
```
